# Tarea 4 - ALU simple de 2 bits

Este proyecto corresponde a una ALU simple de 2 bits implementada en Tinkercad Circuits.

El módulo trabaja con dos entradas binarias de 2 bits, A = A1A0 y B = B1B0. Permite visualizar operaciones lógicas AND, OR, XOR y una suma binaria. Las salidas se muestran mediante LEDs.

## Componentes utilizados

* 74HC08 para operación AND
* 74HC32 para operación OR
* 74HC86 para operación XOR
* 74HC283 para suma binaria
* Interruptores para las entradas
* LEDs como indicadores de salida
* Resistencias
* Protoboard
* Fuente de 5V

## Pruebas realizadas

| Prueba | A  | B  | Resultado esperado           |
| ------ | -- | -- | ---------------------------- |
| 1      | 00 | 00 | AND=0, OR=0, XOR=0, SUMA=000 |
| 2      | 01 | 01 | AND=1, OR=1, XOR=0, SUMA=010 |
| 3      | 10 | 01 | AND=0, OR=1, XOR=1, SUMA=011 |
